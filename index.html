❓ FAQ & Troubleshooting

FlashMind adalah aplikasi flashcard untuk mahasiswa farmasi. Menggunakan Spaced Repetition System (SRS) agar kamu bisa menghafal materi dengan lebih efektif dan efisien.
  1. Buka tab Decks
  2. Pilih topik → pilih subtopik
  3. Pilih mode: Flashcards (rekomendasi), Learn, atau Test
  4. Mulai review dan nilai setiap kartu
Mode Learn & Test hanya tersedia untuk user Premium. Free user hanya bisa menggunakan Flashcards.
Free user dibatasi 5 deck. Deck selebihnya terkunci (🔒). Upgrade ke Premium untuk akses semua deck.
  • Pastikan menyelesaikan review (jangan exit di tengah)
  • Pastikan koneksi internet stabil
  • Coba logout dan login kembali
  • Jika masih bermasalah, hubungi admin
Progress tersimpan otomatis ke cloud. Pastikan login dengan email yang sama di perangkat baru.
  • Gunakan huruf kapital semua
  • Jangan ada spasi di awal/akhir
  • Kode hanya bisa dipakai sekali
  • Hubungi admin jika tetap tidak bisa
Di halaman login, klik "Forgot Password?". Masukkan email, link reset akan dikirim. Cek folder spam jika tidak muncul.
  • Tutup tab lain yang berat
  • Refresh halaman
  • Clear cache browser
  • Gunakan Chrome/Edge (bukan Safari)
  • Logout dan login kembali
  • Pastikan internet stabil
  • URL gambar mungkin tidak valid – hubungi admin
Hubungi admin: adm.pharmacore@gmail.com
